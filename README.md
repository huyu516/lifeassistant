扫码下载

## 功能
- 任务：记录每天要完成的任务和查看已经完成的任务
- 通讯录：可以打电话，最近联系频繁的人排在最上面
- 女性日历：计算女友生理周期(固定29天)，并给予提示
- 开支记录：记录最近在零食，吃饭，购物上的花销并画出图表
